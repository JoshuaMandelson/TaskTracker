A simple iOS task manager built with SwiftUI and SwiftData.

Features

Add and delete tasks
Tap to mark tasks complete/incomplete
Automatic data persistence with SwiftData

Tech Stack

SwiftUI
SwiftData
MVVM Architecture

Requirements

iOS 17.0+
Xcode 15.0+

Usage

Type a task and tap "Add"
Tap any task to toggle completion
Swipe left to delete
